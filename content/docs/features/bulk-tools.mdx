## Overview

Bulk tools let you upload multiple **images** or multiple **videos** in one go, apply a single caption and schedule to all of them, then create scheduled posts in one click. The same two-step flow is used for both **Bulk Video Scheduling** and **Bulk Image Scheduling**: choose accounts and upload files on the left, then set caption and schedule on the right.

Bulk tools are available on **Growth** and **Pro** plans. If your plan doesn’t include them, you’ll see an option to upgrade when you open the tool.

## Where to find it

- **Dashboard** → **Create** → **Bulk tools** (or **More** → **Bulk tools**)
- From there, choose **Bulk Video Upload** or **Bulk Image Upload**

---

## Page layout

Both bulk video and bulk image pages use the same layout:

- **Left:** Account selection (“Post to”), upload zone, and the list of your uploaded items (videos or images).
- **Right:** **Bulk Schedule Settings** - bulk caption, schedule options, and the final “Schedule all” action.

---

## Left panel

### Post to

- **Select all** - Toggle all connected accounts on or off.
- **Remember** - Save your account selection for the next time you use this bulk tool (video and image each have their own remembered selection).
- **Search** - Filter accounts by name or platform.

Only accounts that support the content type (video or image) are shown. Pick the accounts that should receive every post in this batch.

### Upload area

- **Videos:** Click or drag and drop. Up to **40 videos**, **250MB total** per batch. Supported formats: MP4, MOV, AVI, WebM. The 250MB total is enforced in the browser.
- **Images:** Click or drag and drop. Up to **50 images per session**, **250MB total** per batch. Supported formats: JPEG, PNG, WebP, GIF.

Below the upload area you’ll see your items listed (e.g. “Your Videos (0)”). You can adjust the order, edit each item’s caption, and set or change each item’s scheduled time before applying the bulk schedule.

---

## Right panel: Bulk Schedule Settings

### 1. Bulk Caption

- A single caption (up to 2,200 characters) that you can **apply to all** videos or images at once. Use **Apply Caption to All Videos** (or **Images**) to fill every item with this text.
- You can edit captions **per post** in the list on the left - the bulk caption is just a shortcut; each item keeps its own caption and you can change it before scheduling.

### 2. Schedule Settings

- **Start date** - First day to post (e.g. today or a future date).
- **Start time** - Time of the first post on that day.
- **Videos per day** (or **Images per day**) - How many posts per day (1–24). For example, 4 means four posts spread across the day.
- **Time between posts (hours)** - When you choose more than one per day, this sets the gap (e.g. 2 hours). The preview explains how posts are spread “within same day.”
- **Apply This Schedule** - Applies this start date, time, and spacing to every item in the list. A short preview of the schedule is shown (e.g. “Daily schedule (2h apart): 09:00 → 11:00 → 13:00 → 15:00”).

### 3. Confirm & Schedule All

- Review the list on the left (accounts, items, captions, and times).
- Click **Schedule All N Videos** (or **Schedule All N Images**) to create one scheduled post per item for each selected account. You’ll be redirected to the scheduled posts list when done.

## Limits and formats

|                     | Bulk Video           | Bulk Image            |
| ------------------- | -------------------- | --------------------- |
| Max items per batch | 40 videos            | 50 images per session |
| Total batch size    | 250MB                | 250MB                 |
| Caption length      | 2,200 chars (shared) | 2,200 chars (shared)  |
| Formats             | MP4, MOV, AVI, WebM  | JPEG, PNG, WebP, GIF  |

Platform limits (e.g. max file size or aspect ratio per network) still apply. Make sure each file meets the requirements for the platforms you selected.

---

## After scheduling

- All created posts appear under **Dashboard** → **Posts** → **Scheduled**.
- You can edit or cancel any of them from there.

---

## FAQ

**I don’t see Bulk tools.**  
Bulk tools are on Growth and Pro plans. Go to **Dashboard** → **Billing** to upgrade.

**Can I use different captions per video or image?**  
Yes. The right-side bulk caption is only for applying one caption to all items at once. Each item has its own caption; you can edit any of them in the list on the left before scheduling.

**Can I use different times per item?**  
Set the overall schedule (start date, start time, posts per day, gap), then use **Apply This Schedule**. To change a single item’s time, use the date/time on that item’s card in the list before scheduling.
