## Overview

Bulk tools let you upload multiple images or multiple videos in one go and schedule them (e.g. one per day or one per slot). That saves time when you have a batch of content to post over time. Bulk tools are available on **Growth** and **Pro** plans. If your plan doesn’t include them, you’ll see an option to upgrade when you try to open the tool.

## How to use bulk image upload

1. Go to **Dashboard** → **More** → **Bulk tools** (or the equivalent in your app).
2. Open **Bulk Image Upload**.
3. Upload several images (drag and drop or select files).
4. Add a caption if you want (it can apply to all or you can set per image if the tool allows).
5. Choose which accounts should receive the posts.
6. Set a schedule—e.g. one post per day at 9 AM, or assign them to queue slots.
7. Confirm. Each image becomes its own scheduled post.

## How to use bulk video upload

1. Go to **Dashboard** → **More** → **Bulk tools**.
2. Open **Bulk Video Upload**.
3. Upload several videos (one per post).
4. Add captions if you want.
5. Choose which accounts should receive the posts (only video-capable platforms will appear).
6. Set the schedule (e.g. one video per day or per slot).
7. Confirm. Each video becomes its own scheduled post.

## Where bulk tools live

- **Dashboard** → **More** (or sidebar) → **Bulk tools** — From here you can open **Bulk Image Upload** or **Bulk Video Upload**.

If you don’t see Bulk tools, your plan may not include them. Check **Billing** to upgrade to Growth or Pro.

## Tips

- Make sure each file (image or video) meets the usual size and format rules for the platforms you pick.
- You can go to **Posts** after and see all the scheduled posts; you can edit or cancel any of them.

## Common questions

**Q: I don’t see Bulk tools. Why?**  
A: Bulk tools are on Growth and Pro plans. Go to **Dashboard** → **Billing** to upgrade.

**Q: Can I use different captions for each image/video?**  
A: It depends on the bulk tool screen. If there’s a per-item caption field, you can; otherwise one caption may apply to all. Check the form when you upload.
