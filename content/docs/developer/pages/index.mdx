# Pages Reference

This section contains per-page documentation for the Next.js app router frontend. Each page doc describes **Route**, **Purpose**, **Access**, **Data flow**, **Components**, **State**, **Key business logic**, **URL params**, **Error states**, and **Related pages**.

The doc agent writes `.mdx` files here (one per route). Current pages include:

- Landing, Auth, Onboarding
- Dashboard (redirect), Composer, Create (type selector and create by type)
- Posts list, Post detail, Post edit, Scheduled, Drafts, Posted
- Connections, Connection select (LinkedIn, Instagram, Facebook)
- Settings, Billing, Feedback, Calendar, More
- Bulk tools (image, video), Teams, API Keys
- Privacy, Terms

Run the doc agent to generate or update the individual page files under this folder.
