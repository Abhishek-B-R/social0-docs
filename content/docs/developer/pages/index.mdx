## Overview

The rest of these docs cover every part of Social0 you use: signing up, connecting accounts, creating each type of post, scheduling, queue, bulk tools, billing, and the Dashboard (Composer, Posts, Connections, Settings, and more). Use the sidebar or the [main docs index](/docs) to jump to the topic you need.

## Where to go for what

- **New to Social0?** - Start with [Welcome](/docs) and [Getting started](/docs/getting-started).
- **Connecting an account?** - [Connecting accounts](/docs/getting-started/connecting-accounts) and the [Platforms](/docs/platforms) section (e.g. [Twitter](/docs/platforms/twitter), [Instagram](/docs/platforms/instagram)).
- **Creating a post?** - [Composer](/docs/dashboard/composer), [Create](/docs/dashboard/create), and [Post types](/docs/post-types) (text, image, video, thread, collection).
- **Scheduling or queue?** - [Scheduling](/docs/features/scheduling) and [Queue](/docs/features/queue).
- **Plans and billing?** - [Billing](/docs/billing) and [Plans](/docs/billing/plans).
- **Dashboard screens?** - [Dashboard](/docs/dashboard), [Posts](/docs/dashboard/posts), [Connections](/docs/dashboard/connections), [Settings](/docs/dashboard/settings).

If you can’t find what you need, use **Feedback** in the app or contact support.
