## Overview

Social0 offers two integration paths:

- **[MCP Server](/docs/integrations/mcp)** — official Model Context Protocol server for Claude, Cursor, ChatGPT, and VS Code
- **[REST API](/docs/api)** — full programmatic access at `https://api.social0.app/v1`

Both use [API keys](/docs/dashboard/api-keys) from Developer settings.

## MCP Server (recommended for AI users)

Manage posts with natural language in your AI assistant:

```text
Show my connected Social0 accounts.
Create a LinkedIn post about our product launch.
Publish my latest draft.
```

[Get started with MCP →](/docs/integrations/mcp/quickstart)

## REST API (for developers)

Build scripts, backends, and custom integrations:

```bash
curl https://api.social0.app/v1/accounts \
  -H "Authorization: Bearer sk_live_YOUR_KEY"
```

[API Quickstart →](/docs/api/quickstart)

## Tips

- Connect platforms in [Connections](/docs/dashboard/connections) before using MCP or API
- Use [webhooks](/docs/api/webhooks) to avoid polling publish status
- One API key per integration (Claude vs Cursor vs CI)
