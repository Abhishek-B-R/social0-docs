## Overview

These docs are written for people who use Social0 to create and schedule posts. If you’re looking for ways to connect Social0 to other tools (e.g. Zapier, your own app) or to use it via an API, this page points you in the right direction.

## What’s available today

- **Dashboard and Composer** — Create, schedule, and publish posts from the Social0 app (requires a paid plan and connected accounts). On the free tier you can only explore the dashboard; you cannot connect accounts or post.
- **Connections** — Add or remove social accounts from the Connections page (paid plans only). Each platform uses its official sign-in; we only ask for the permissions we need to post.

There is no public API or developer docs for building custom integrations yet. We’re working on it.

## What’s coming

- **API keys** — A way to use Social0 from other tools or scripts (e.g. post from a spreadsheet or automation). When it’s ready, you’ll manage API keys from the Dashboard and we’ll document how to use them.
- **Integrations** — Pre-built connections to popular tools. We’ll announce these when they’re available.

## What you can do now

- **Use the product** — Everything in these docs applies to the app today: getting started, connecting accounts, post types, scheduling, queue, billing, and so on.
- **Send feedback** — Use **Feedback** in the Dashboard to suggest integrations or API features you’d like. That helps us prioritize.
- **Contact us** — If you need API access, a custom integration, or a partnership before we launch it publicly, reach out through the contact details on the Social0 website or via the Feedback page. We’ll get back to you.

## Tips

- Don’t share your Social0 password with third-party tools. When API keys are available, you’ll use those instead and can revoke them anytime.
- Bookmark this page or the main docs index so you can quickly find help for the product.
