## Overview

Social0 has a **versioned public REST API** at `https://api.social0.app/v1`. Use it to publish, schedule, upload media, and receive webhook events from your backend, CI pipeline, or automation tools.

## What's available today

- **[REST API](/docs/api)** - Full programmatic access with API keys (`sk_live_…`)
- **[Quickstart](/docs/api/quickstart)** - Publish your first post in 5 minutes
- **[Developer settings](/docs/dashboard/api-keys)** - Create API keys and manage webhooks
- **Dashboard and Composer** - Create, schedule, and publish posts from the app
- **Connections** - Add or remove social accounts (required before API publishing)

## Get started with the API

1. Connect at least one account in [Dashboard → Connections](/docs/dashboard/connections)
2. Create an API key in [Developer settings](/docs/dashboard/api-keys)
3. Follow the [Quickstart](/docs/api/quickstart) to publish your first post

## What's coming

- **Pre-built integrations** - Zapier, Make, and other automation platforms
- **Official SDKs** - Node.js and Python client libraries
- **API key scopes** - Fine-grained permissions per key

## Tips

- Don't share your Social0 password with third-party tools — use API keys instead and revoke them anytime.
- Use [webhooks](/docs/api/webhooks) instead of polling for publish status.
- Send feedback via **Dashboard → Feedback** to suggest integrations you'd like.
