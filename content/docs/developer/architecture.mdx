## Overview

You create a post in the Composer or Create flow, pick your accounts, and choose Publish now or Schedule. Social0 then sends that post to each platform you selected. For scheduled posts, we publish at the time you set (using your timezone from Settings). For queues, we use the time windows you defined in Settings → Queue.

We don’t publish internal technical docs (e.g. system design or infrastructure). This documentation is about how *you* use the product. If you need API access or custom integrations, see [Integrations and API](/docs/developer) and contact us.

## Tips

- Your content and account connections are used only to run the product (posting, scheduling, showing your posts list). We don’t sell your data.
- If something doesn’t work (e.g. a post failed), check **Connections** for expired accounts and **Billing** for an active plan, then retry from the Posts list.
