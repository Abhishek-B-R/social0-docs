## Overview

Social0 integrates with external tools in two ways:

- **[MCP](/docs/integrations/mcp)** — manage posts from Claude.ai, ChatGPT, Cursor, and other AI hosts using natural language
- **[REST API](/docs/api)** — programmatic access for scripts, backends, and custom apps

## MCP (AI assistants)

**Remote URL (recommended):** add `https://mcp.social0.app/mcp` in any AI that supports remote MCP connectors, then authorize with Social0 OAuth. No Node.js. No API key in config.

**Local stdio (fallback):** `npx -y social0-mcp` with a `sk_live_` key when your host only supports stdio.

[Get started with MCP →](/docs/integrations/mcp/quickstart)

## REST API

Full programmatic access at `https://api.social0.app/v1`. Authenticate with `Authorization: Bearer sk_live_…`.

- [API Overview](/docs/api) — complete reference, guides, and OpenAPI spec
- [Quickstart](/docs/api/quickstart) — publish your first post in 5 minutes
- [Interactive docs](https://api.social0.app/docs) — live Swagger UI

## Prerequisites

1. Social0 account with at least one [connected platform](/docs/dashboard/connections)
2. For remote MCP: nothing else — OAuth mints a connector key
3. For local MCP or REST: an [API key](/docs/dashboard/api-keys) (`sk_live_…`)
