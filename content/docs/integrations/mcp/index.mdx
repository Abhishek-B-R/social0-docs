## Overview

Social0 MCP lets AI assistants create, schedule, and publish posts to your connected social accounts in natural language.

**Remote / hosted OAuth is the default.** Point any AI that accepts a remote MCP connector URL at Social0 — no Node.js, no API key in config.

| | **Remote (recommended)** | **Local (`npx`)** |
|--|--------------------------|-------------------|
| Endpoint | `https://mcp.social0.app/mcp` | `npx -y @social0/mcp` |
| Auth | Social0 OAuth (Connect → approve) | `SOCIAL0_API_KEY` in config |
| Needs Node.js? | No | Yes (Node 20+) |
| Best for | Claude.ai, ChatGPT, Cursor (HTTP MCP), and any host with remote connectors | Cursor / Claude Desktop / VS Code when only stdio is available |

Both use the same tools and the same Social0 publish pipeline as the dashboard.

Prefer the [CLI](/docs/integrations/cli) (`npm install -g social0`) for terminals, CI, and AI agents with shell access. Use MCP when the host only supports MCP connectors (no reliable shell).

## Prerequisites

1. A [Social0 account](https://social0.app)
2. At least one platform connected in [Dashboard → Connections](/docs/dashboard/connections)

API keys are only for the local `npx` path — create them in [Developer settings](/docs/dashboard/api-keys). Hosted OAuth mints a connector key automatically (you can revoke it there anytime).
