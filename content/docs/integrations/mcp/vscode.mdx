## Prefer remote first

If your VS Code MCP extension supports a remote/HTTP MCP URL, use:

```text
https://mcp.social0.app/mcp
```

Authorize with Social0 OAuth when prompted. See the [MCP quickstart](/docs/integrations/mcp/quickstart).

## Local npx (stdio)

Most VS Code MCP setups use stdio:

```json
{
  "mcpServers": {
    "social0": {
      "command": "npx",
      "args": ["-y", "@social0/mcp"],
      "env": {
        "SOCIAL0_API_KEY": "sk_live_your_key_here"
      }
    }
  }
}
```

Requires [Node.js 20+](https://nodejs.org/) and an API key from [Developer settings](/docs/dashboard/api-keys). Config file location varies by extension (often `.vscode/mcp.json` or user settings).

Reload VS Code, then ask: “Show my connected Social0 accounts.”
