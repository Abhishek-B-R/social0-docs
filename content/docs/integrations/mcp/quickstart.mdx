## Option A — Remote URL (recommended)

Works with **Claude.ai**, **ChatGPT**, **Cursor** (if HTTP MCP is supported), and any other host that accepts a remote MCP URL.

1. Connect social accounts in [Dashboard → Connections](https://social0.app/dashboard/connections).
2. Open your AI app’s connectors / MCP settings:
   - **Claude.ai** → Settings → Connectors → Add custom connector
   - **ChatGPT** → Settings → Connectors (wording may vary)
   - Other hosts → look for remote MCP / custom connector
3. Add this URL:

```text
https://mcp.social0.app/mcp
```

4. Click **Connect** and authorize on social0.app (OAuth PKCE).
5. Ask: “Show my connected Social0 accounts.”

No Node.js. No `SOCIAL0_API_KEY` in config. OAuth creates a connector API key (the UI may show “Claude MCP Connector”). Revoke it anytime in [Dashboard → API keys](/docs/dashboard/api-keys); reconnecting mints a new one.

### Cursor (remote URL)

If your Cursor build supports HTTP MCP servers:

```json
{
  "mcpServers": {
    "social0": {
      "url": "https://mcp.social0.app/mcp"
    }
  }
}
```

Authorize when prompted. If remote/OAuth isn’t available, use Option B.

### Media tip (remote)

Use `upload_media` with a public `url` or base64 `data`. Do **not** pass `file_path` — sandbox paths fail on the hosted server.

## Option B — Local npx (fallback)

Use this for stdio-only hosts (Cursor / Claude Desktop / VS Code, and ChatGPT Desktop only if remote isn’t available).

1. Install [Node.js 20+](https://nodejs.org/).
2. Create an API key at [Dashboard → API keys](https://social0.app/dashboard/api-keys) (`sk_live_…`).
3. Paste this into your host MCP config:

```json
{
  "mcpServers": {
    "social0": {
      "command": "npx",
      "args": ["-y", "@social0/mcp"],
      "env": {
        "SOCIAL0_API_KEY": "sk_live_your_key_here"
      }
    }
  }
}
```

4. Reload MCP / restart the app.
5. Ask: “Show my connected Social0 accounts.”

### Where to paste

| Host | Location |
|------|----------|
| Cursor | Settings → MCP, or `.cursor/mcp.json` |
| Claude Desktop | `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) |
| VS Code | Copilot / MCP settings (stdio) |
| ChatGPT Desktop | Prefer Option A; local only if remote isn’t available |
