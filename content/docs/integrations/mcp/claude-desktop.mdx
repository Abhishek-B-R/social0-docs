## Prefer remote first

On **Claude.ai** (web), use Settings → Connectors → Add custom connector with:

```text
https://mcp.social0.app/mcp
```

Click **Connect** and authorize on social0.app. No Node.js or API key in config. See the [MCP quickstart](/docs/integrations/mcp/quickstart).

## Local npx (Claude Desktop)

Use this when you need the desktop app with stdio MCP.

1. Install [Node.js 20+](https://nodejs.org/).
2. Create an API key in [Developer settings](/docs/dashboard/api-keys).
3. Edit the Claude Desktop config:

| OS | Path |
|----|------|
| macOS | `~/Library/Application Support/Claude/claude_desktop_config.json` |
| Windows | `%APPDATA%\Claude\claude_desktop_config.json` |
| Linux | `~/.config/Claude/claude_desktop_config.json` |

```json
{
  "mcpServers": {
    "social0": {
      "command": "npx",
      "args": ["-y", "social0-mcp"],
      "env": {
        "SOCIAL0_API_KEY": "sk_live_your_key_here"
      }
    }
  }
}
```

4. Fully quit and reopen Claude Desktop.
5. Ask: “Show my connected Social0 accounts.”
