## Overview

The Social0 REST API at `https://api.social0.app/v1` is the underlying interface for the [CLI](/docs/integrations/cli), [MCP server](/docs/integrations/mcp), scripts, and custom apps. Authenticate with `Authorization: Bearer sk_live_ŌĆ”`.

**Higher-level options:** Always prefer the [`social0` CLI](/docs/integrations/cli) when you have a terminal or agent shell. Use the [MCP Server](/docs/integrations/mcp) **only if you canŌĆÖt use the CLI** (chat-only hosts). Call REST directly for custom backends.
