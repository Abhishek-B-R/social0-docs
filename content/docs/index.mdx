# Social0 Docs

One composer. Multiple platforms. No copy-paste, no tab-switching.

Social0 lets you create and schedule posts for **Twitter/X**, **Instagram**, **LinkedIn**, **YouTube**, **TikTok**, **Bluesky**, **Threads**, **Facebook**, and **Pinterest** from a single dashboard.
