# Post Types

- **Text** — Plain text; supported on Facebook, Bluesky, Twitter/X, LinkedIn, Threads.
- **Image** — Image with optional caption; supported on Facebook, Bluesky, Twitter/X, LinkedIn, Threads, Pinterest, TikTok, Instagram.
- **Video** — Video with optional caption; supported on Facebook, Bluesky, Twitter/X, LinkedIn, Threads, YouTube, Pinterest, TikTok, Instagram.
- **Thread** — Multi-post thread (Twitter/Threads/Bluesky style); supported on Twitter/X, Threads, Bluesky.
- **Collection** — Multiple images/videos in one post (e.g. carousel); supported on Twitter/X, Threads, Instagram.

You can start in the **Composer** with freeform text and media; we route you to the right form (text, image, video, threads, collection) based on what you add.
