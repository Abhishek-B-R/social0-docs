## Overview

Social0 supports five post types: text, image, video, thread, and collection. When you use the Composer, we suggest a type based on what you add (text only, images, video, or multiple items). You can also go to **Create** and choose the type yourself.

## Post types at a glance

- **Text** - Plain text. Good for Twitter/X, LinkedIn, Threads, Facebook, Bluesky.
- **Image** - One or more images with an optional caption. Works on most platforms.
- **Video** - One video with an optional caption. Works on YouTube, TikTok, Instagram, and others.
- **Thread** - Multiple posts in a row (like a Twitter thread). For Twitter/X, Threads, and Bluesky.
- **Collection** - One post with multiple images or videos (collection post). For Twitter/X, Threads, and Instagram.

Not every platform supports every type. When you pick accounts for a post, we only show platforms that support that type.

## How to choose a post type

1. **Quick way:** Open the **Composer**, type or add media, and click **Continue**. We’ll pick the right type and take you to the next step.
2. **Manual way:** Go to **Create**, then choose **Text**, **Image**, **Video**, **Threads**, or **Collection** and fill in the form.

## Tips

- Start in the Composer if you’re not sure; it’s the fastest way to get from idea to publish.
- For threads, use the Thread type so each part posts in order. For a single post with several images, use Collection.
