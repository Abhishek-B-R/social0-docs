## Overview

An image post is one or more photos with an optional caption. You add the images, write the caption if you want, pick which accounts get it, and publish or schedule. It works on Facebook, Bluesky, Twitter/X, LinkedIn, Threads, Pinterest, TikTok, and Instagram.

## How to create an image post

1. Open the **Composer** and add one or more images (drag and drop or click to upload). Add a caption if you like.  
   **Or** go to **Create** → **Image post** and upload images there.
2. Click **Continue** (if you started in the Composer).
3. Select the accounts that should receive the post.
4. Click **Publish now** to post right away, or **Schedule** to choose a date and time.

You can reorder images by dragging. Each platform has its own rules for how many images and how big they can be; we’ll guide you if something isn’t supported.

## Where image posts work

- Facebook, Bluesky, Twitter/X, LinkedIn, Threads, Pinterest, TikTok, Instagram

## Tips

- Use a clear, high-quality image so it looks good on every platform you pick.
- Captions can include hashtags; add them in the caption field when you create the post.

## Common questions

**Q: How many images can I add?**  
A: It depends on the platform. When you select accounts, we only show platforms that support the number of images you’ve added. For a single image, all supported platforms work.

**Q: What if my image is too big?**  
A: We’ll tell you if a file is too large. Compress or resize the image and try again.
