## Overview

A **collection post** is one post that contains **multiple videos** or a **mix of images and videos** in the same post (e.g. one tweet or one Instagram post with several pieces of media). You add all the items, add an optional caption, pick which accounts get it, and publish or schedule.

**When to use which:**

- **Multiple images only** → Use the **Image** post type (image form).
- **Single video** → Use the **Video** post type (video form).
- **Multiple videos in one post**, or **images and videos together in one post** → Use **Collection** (this page).

Collection posts work on Twitter/X, Threads, and Instagram. Other platforms (e.g. LinkedIn, YouTube) don’t support this format in Social0; use a single image or video post for those.

## How to create a collection post

1. Open the **Composer** and add **multiple** images and/or videos (drag and drop or click to upload).  
   **Or** go to **Create** → **Collection** and add them there.
2. Add a caption if you like. You can reorder items by dragging.
3. Click **Continue** (if you started in the Composer).
4. Select the accounts that should receive the post (only Twitter/X, Threads, and Instagram will appear for this type).
5. Click **Publish now** or **Schedule**.

All items are published together as one collection post.

## Where collection posts work

- **Twitter/X** - Multiple images or videos in one tweet
- **Threads** - Multiple images/videos in one post
- **Instagram** - Multiple images/videos in one post

## Tips

- Order matters: the first image or video is what people see first. Drag to reorder before publishing.
- Mix images and videos in one collection when the platform allows it.

## Common questions

**What’s the difference between Collection and Thread?**  
A **collection post** is one post with multiple images/videos. A **thread** is multiple separate posts that go out in order (e.g. tweet 1, tweet 2, tweet 3).

**When do I use Collection vs Image or Video?**  
Use **Image** for multiple images only. Use **Video** for a single video. Use **Collection** for multiple videos in one post, or a mix of images and videos in the same post.

**How many items can I put in a collection post?**  
It depends on the platform (e.g. Instagram has a limit per post). We’ll only show platforms that support the number of items you added.
