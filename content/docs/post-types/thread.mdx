# Thread

Create a multi-post thread. Supported on **Twitter/X**, **Threads**, and **Bluesky**. Each post in the thread can have text and up to 4 attachments (images/videos).

Create via **Composer** by toggling "Add another post as thread" and adding multiple posts, or **Create → Manual setup → Threads**.
