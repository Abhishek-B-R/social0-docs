## Overview

A thread is a series of posts that go out one after another—like a Twitter thread or a Threads thread. Each part can have text and up to four images or videos. Threads in Social0 work for Twitter/X, Threads, and Bluesky.

## How to create a thread

1. Open the **Composer** and turn on **Add another post as thread** (or go to **Create** → **Threads**).
2. Write the first post. Add images or video if you want (up to four per post).
3. Click to add the next post. Fill in text and optional media for each part.
4. Click **Continue** (if you started in the Composer).
5. Select the accounts that should receive the thread (only Twitter/X, Threads, and Bluesky will appear).
6. Click **Publish now** or **Schedule**.

We’ll post each part in order. The first post goes out, then the second, and so on.

## Where thread posts work

- **Twitter/X** — Multi-tweet threads  
- **Threads** — Multi-post threads  
- **Bluesky** — Multi-post threads  

Instagram, LinkedIn, etc. don’t support this format in Social0; use a single text, image, or video post for those.

## Tips

- Keep each part focused. Short posts often work better for threads.
- You can reorder parts by dragging before you publish or schedule.

## Common questions

**Q: How many posts can I add to a thread?**  
A: You can add as many parts as you need. Very long threads may take a while to publish because we send each part in order.

**Q: Can I schedule a thread?**  
A: Yes. The time you pick is when the first post goes out; the rest follow after that.
