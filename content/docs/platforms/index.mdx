# Platforms

Social0 supports:

- **Twitter/X** — Text, image, video, thread, collection
- **Bluesky** — Text, image, video, thread
- **LinkedIn** — Text, image, video (personal and company pages)
- **Threads** — Text, image, video, thread, collection
- **Instagram** — Image, video, collection (via Facebook Page connection)
- **Facebook** — Text, image, video (Pages)
- **YouTube** — Video
- **TikTok** — Image, video
- **Pinterest** — Image, video

Platform IDs in the app: `twitter_x`, `bluesky`, `linkedin`, `threads`, `instagram`, `facebook`, `youtube`, `tiktok`, `pinterest`.
