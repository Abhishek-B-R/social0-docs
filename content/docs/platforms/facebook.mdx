## Overview

Social0 posts to **Facebook Pages**, not personal profiles. You can connect one or more Pages from **Dashboard** → **Connections** and publish text, images, and videos to them.

## How to connect a Facebook Page

1. Go to **Dashboard** → **Connections**.
2. Find **Facebook** and click **Connect**.
3. Sign in to Facebook if needed and approve the permissions.
4. Choose which **Page(s)** you want to connect. You can select more than one.
5. You’ll return to Social0 with those Pages connected.

When you create a post, you pick which connected Page (or Pages) should receive it.

## What you can post

- **Text** - Plain text posts.
- **Image** - One or more images with an optional caption.
- **Video** - Video with optional caption.

## Tips

- You must be an admin or have the right role on a Page to connect it. If a Page doesn’t appear, check your role in Facebook Page settings.
- Connecting a Page that’s linked to Instagram does not automatically connect Instagram; connect Instagram separately if you want to post there too.
