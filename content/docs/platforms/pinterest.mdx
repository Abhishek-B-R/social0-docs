## Overview

You can connect your Pinterest account to Social0 and publish images and videos. Connection is done from **Dashboard** → **Connections**. You’ll choose which board to pin to when you set up the connection (or when required by Pinterest).

## How to connect Pinterest

1. Go to **Dashboard** → **Connections**.
2. Find **Pinterest** and click **Connect**.
3. Sign in to Pinterest when asked and approve the permissions.
4. If Pinterest asks you to pick a board, choose one (you can change which board a pin goes to when you create the post, if the product allows).
5. You’ll return to Social0 with the account connected.

When you create an image or video post, you can select your Pinterest account to receive it.

## What you can post

- **Image** - One or more images with an optional caption/description.
- **Video** - Video with optional caption.

## Tips

- Pins work best with a clear image and a short, descriptive caption. Add links and details in the post when you create it.
- If you have multiple boards, the connection flow or the post creation may ask you to choose where to pin.
