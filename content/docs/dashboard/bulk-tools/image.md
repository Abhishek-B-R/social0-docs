## Overview

**Bulk image scheduling** lets you upload many images at once, pick which connected accounts they go to, add a caption (for all or per image), and set a schedule. Each image becomes its own post. The page has two main areas: on the left you choose accounts and manage your images; on the right you set bulk caption and schedule, then confirm. This feature is in **Beta** and is available on **Growth** and **Pro** plans.

## How to use bulk image scheduling

1. Go to **Dashboard** → **Bulk tools** (or **More** → **Bulk tools** on mobile) and open **Bulk Image Upload**.
2. **Post to** - Select the accounts that should receive the posts. Use **Select all** to tick every account, **Search accounts** to filter, and **Remember** if you want your choice saved for next time. You must select at least one account before you can schedule.
3. **Upload images** - Click or drag and drop into the upload zone. You’ll see how many files you’ve added and the total size (e.g. 2 files / 50 files, 839 KB / 250 MB). Reorder images with the up arrow; remove one with the trash icon.
4. **Your images** - Each image has its own **Image caption** (optional), **date**, and **time**. Use **Collapse all** / **Expand all** to show or hide details. You can leave dates and times blank and set them in bulk in the right panel.
5. **Bulk Schedule Settings** (right panel):
   - **Bulk caption** - Enter a caption and click **Apply Caption to All Images** to copy it to every image.
   - **Schedule settings** - Set **Start date**, **Start time**, and **Images per day** (1–24). Click **Apply This Schedule** to fill each image’s date and time. The summary shows how they’re spread (e.g. “Daily schedule (2h apart): 01:50”, “Total duration: ~2 days”).
6. **Confirm & schedule** - Review the list and any per-image changes, then click **Schedule All X Images**. If you haven’t selected any accounts, you’ll see a message asking you to select at least one.

Scheduled images appear in **Posts** as **Scheduled** (or **Queued** if you used queue slots).

## What you’ll see

- **Post to** - Grid of connected accounts (avatar and platform). Select which ones get the posts. Accounts with connection issues may be greyed out.
- **Upload zone** - File count and size limits (e.g. up to 50 files, 250 MB total). Click or drag to add more.
- **Your images** - One row per image with caption, date, time, reorder, and delete. Expand or collapse all to work faster.
- **Bulk Schedule Settings** - Bulk caption and “Apply Caption to All Images”; start date, start time, images per day, and “Apply This Schedule”; then “Schedule All X Images” at the bottom.

## Tips

- Use **Apply Caption to All Images** and **Apply This Schedule** first, then tweak individual images if needed.
- Set **Images per day** to control how many posts go out each day; the app spreads them from the start time.
- Use clear file names so you can tell images apart in the Posts list later.

## Common questions

**Q: What if I have dozens of images?**  
A: Upload as many as the limit allows (e.g. 50 files, 250 MB). Set the schedule so they go out over time (e.g. one or more per day). You can edit or delete any scheduled post from the Posts list.

**Q: Do I have to pick the same accounts for every image?**  
A: In this flow you choose one set of accounts for all images. To send some images to different accounts, schedule the first batch, then create another bulk run or edit individual posts in the Posts list.

**Q: Why does it say “Please select at least one account”?**  
A: You need to select at least one connected account under **Post to** before **Schedule All** will work. Tick the accounts you want and try again.
