## Overview

When you open **Create**, you see five options: Text, Image, Video, Threads, and Collection. Each opens a form for that type of post. You fill it in, pick your accounts, and publish or schedule. You can also land on one of these forms from the Composer after clicking Continue (we’ll have chosen the type for you).

## How to create each type

**Text** — Enter your text. Pick accounts. Publish or schedule. No media.

**Image** — Upload one or more images, add a caption, pick accounts, then publish or schedule.

**Video** — Upload one video, add a caption, pick accounts (video-capable only), then publish or schedule.

**Threads** — Add the first post (text and optional media), then add more posts. Pick accounts (Twitter/X, Threads, Bluesky). Publish or schedule; we’ll send each part in order.

**Collection** — Upload multiple images and/or videos, add a caption, pick accounts (Twitter/X, Threads, Instagram), then publish or schedule.

## Tips

- If you have a draft or a post you’re editing, you may see your previous content pre-filled. Change it as needed before publishing or scheduling.
- Each form shows only the accounts that support that post type. If an account is missing, it doesn’t support that type (e.g. YouTube only for video).
