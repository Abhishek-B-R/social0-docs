## Overview

**Create** is the “do it myself” way to start a post: you choose the type (Text, Image, Video, Threads, or Collection) first, then fill in the form. Use this when you know exactly what kind of post you want instead of letting the Composer guess from what you type and add.

## How to create a post by type

1. In the Dashboard, open **Create** (from the sidebar or menu).
2. Choose the post type:
   - **Text** — Words only.
   - **Image** — One or more images with optional caption.
   - **Video** — One video with optional caption.
   - **Threads** — Multiple posts in a row (Twitter/Threads/Bluesky).
   - **Collection** — Multiple images/videos in one post (collection post).
3. You’ll be taken to the form for that type. Fill in the content (text, upload images/video, etc.).
4. Select which accounts should receive the post.
5. Click **Publish now** or **Schedule** (or save as draft).

## When to use Create vs Composer

- **Composer** — You type and add media, click Continue, and we pick the type. Fastest for most posts.
- **Create** — You pick the type first, then fill the form. Use this when you want full control (e.g. “I want a thread” or “I want a collection”).

## Tips

- From the Create screen you can go back to **Posts** or **Composer** using the links at the top or in the sidebar.
- If you started in the Composer and clicked Continue, you’re already in the same flow as Create—just for the type we chose. You can still edit everything before publishing or scheduling.
