## Overview

You can edit a post as long as it’s still a draft or scheduled. Open the post from the Posts list, then use **Edit** (or “Edit post”). You’ll be taken to the same kind of form you used to create it (text, image, video, thread, or collection), with your current content filled in. Change what you want, then save or reschedule.

## How to edit a draft

1. Go to **Dashboard** → **Posts**.
2. Find the post with status **Draft** and click it.
3. Click **Edit** (or “Edit post”).
4. You’ll see the create form with your text and media. Change the caption, add or remove images/video, or change the accounts.
5. Click **Save** or **Publish now** or **Schedule**.

Your changes are saved. If you publish or schedule, the post will use the new content.

## How to edit a scheduled post

1. Go to **Dashboard** → **Posts**.
2. Find the post with status **Scheduled** (or **Queued**) and click it.
3. Click **Edit** (or “Edit post”).
4. Change the content and/or the scheduled date and time (or queue slot).
5. Save. The post will still go out at the new time you set.

## Tips

- You can’t edit a post after it’s already been published to a platform. What’s live is live. Use “Post again” if you want to send the same (or updated) content again.
- If the post is queued, you can change which slot it’s in when you edit.

## Common questions

**Q: I edited the post but the time didn’t change.**  
A: Make sure you change the date/time (or queue slot) in the form and save. If you only changed the text, the schedule stays the same.

**Q: Can I turn a scheduled post back into a draft?**  
A: Yes. Edit the post and remove the schedule (or set it to “Draft”). It will stay in your list as a draft until you publish or schedule again.
